Welcome to StreamForge

StreamForge is a modern, distributed event streaming platform designed for real-time data processing at scale. Built with Rust for performance and safety, it provides a highly available, fault-tolerant message broker system with strong replication guarantees.

Status: Active Development 🚀 - Current version focuses on core broker functionality, Go SDK, and admin tools.

Key Features

⚡ High Performance

Optimized for millions of messages per second with latency in the single-digit milliseconds.

🔄 Distributed

Multi-broker cluster support with automatic failover and replication across brokers.

🛡️ Fault Tolerant

Strong replication guarantees with configurable min-in-sync replicas for data durability.

📊 Monitoring

Built-in Prometheus metrics and Grafana dashboards for comprehensive observability.

🐳 Container Ready

Single binary deployment with Docker and Kubernetes support out of the box.

💾 Reliable Storage

High-performance C++ storage backend with memory-mapped file I/O for optimal throughput.

Project Structure

StreamForge consists of several integrated components:

  • Broker (Rust) - Core message broker with cluster coordination and replication
  • Storage (C++) - High-performance log engine with memory-mapped file I/O
  • SDK (Go) - Client library for producers and consumers
  • Admin (Go) - CLI and REST API for cluster management
  • Monitoring - Prometheus metrics and Grafana dashboards
